# Bootstrap布局
## Container容器
- Container是Bootstrap中最基本的布局元素，在使用默认网格系统时必不可少。
- 容器用于容纳、填充内容，有时还能使内容剧中。虽然容易可以嵌套，但大多数布局并不需要嵌套容器。
- `.container`或`.container-fluid` 是启用整个栅格系统必不可少的前置条件，
分别对应选择一个响应式的、固定宽度的容器，或者选择一个流式自适应浏览器或容器最大合法宽度的窗口（意味着任何时候它的宽度总是100%）。

Bootstrap提供了三种不同的容器：
- `.container`，在每个响应式断点处设置一个`max-width`。
- `.container-{breakpoint}`，直到指定的断点为止`width: 100%`。
- `.container-fluidwidth: 100%`，在所有断点处都是如此。
