## 1.2. 删除本地标签
```bash
Razer@Razer MINGW64 /d/code/git-test (master)
$ git tag
v0.1
v1.0

Razer@Razer MINGW64 /d/code/git-test (master)
$ git tag -d v1.0
Deleted tag 'v1.0' (was f453e5f)

Razer@Razer MINGW64 /d/code/git-test (master)
$ git tag
v0.1
```

## 1.3. 推送标签
### 1.3.1. 推送某个标签到远程
```bash
$ git push origin v1.0
```

### 1.3.2. 一次性推送全部尚未推送到远程的本地标签
```bash
$ git push origin --tags
```

## 1.4. 删除远程标签
```bash
# 本地删除：
$ git tag -d v0.9

# 远程删除
$ git push origin :refs/tags/v0.9
```
